–û—Ç–ª–∏—á–Ω–æ\! –í—ã –ø–µ—Ä–µ—Ö–æ–¥–∏—Ç–µ –∫ —Å–ª–µ–¥—É—é—â–µ–º—É –≤–∞–∂–Ω–æ–º—É –±–ª–æ–∫—É –≤ —É–ø—Ä–∞–≤–ª–µ–Ω–∏–∏ –ø–æ—Ç–æ–∫–æ–º ‚Äî **–¶–∏–∫–ª–∞–º** (Loops). –¶–∏–∫–ª—ã –ø–æ–∑–≤–æ–ª—è—é—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å –æ–¥–∏–Ω –∏ —Ç–æ—Ç –∂–µ –±–ª–æ–∫ –∫–æ–¥–∞ –º–Ω–æ–≥–æ–∫—Ä–∞—Ç–Ω–æ, —ç–∫–æ–Ω–æ–º—è –≤—Ä–µ–º—è –∏ —É—Å–∏–ª–∏—è –ø—Ä–æ–≥—Ä–∞–º–º–∏—Å—Ç–∞.

---

## üîÅ –¶–∏–∫–ª—ã (Loops)

–ê–≤—Ç–æ—Ä –æ–±—ä—è—Å–Ω—è–µ—Ç, —á—Ç–æ —Ü–∏–∫–ª—ã –Ω—É–∂–Ω—ã –¥–ª—è —Ä–µ—à–µ–Ω–∏—è –∑–∞–¥–∞—á–∏, –∫–æ–≥–¥–∞ –æ–¥–Ω–æ –∏ —Ç–æ –∂–µ –¥–µ–π—Å—Ç–≤–∏–µ (–Ω–∞–ø—Ä–∏–º–µ—Ä, `print("Hello World")`) –Ω—É–∂–Ω–æ –≤—ã–ø–æ–ª–Ω–∏—Ç—å –∑–∞–¥–∞–Ω–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ä–∞–∑. –í–º–µ—Å—Ç–æ —Ç–æ–≥–æ —á—Ç–æ–±—ã –ø–∏—Å–∞—Ç—å –∫–æ–º–∞–Ω–¥—É 10 —Ä–∞–∑ –≤—Ä—É—á–Ω—É—é, –≤—ã –∏—Å–ø–æ–ª—å–∑—É–µ—Ç–µ —Ü–∏–∫–ª.

### 1\. `for` Loop (–¶–∏–∫–ª `for`)

–¶–∏–∫–ª `for` —è–≤–ª—è–µ—Ç—Å—è –Ω–∞–∏–±–æ–ª–µ–µ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–Ω—ã–º —Ç–∏–ø–æ–º —Ü–∏–∫–ª–∞ –∏ –∏–¥–µ–∞–ª—å–Ω–æ –ø–æ–¥—Ö–æ–¥–∏—Ç, –∫–æ–≥–¥–∞ –≤—ã –∑–∞—Ä–∞–Ω–µ–µ –∑–Ω–∞–µ—Ç–µ, —Å–∫–æ–ª—å–∫–æ —Ä–∞–∑ –Ω—É–∂–Ω–æ –ø–æ–≤—Ç–æ—Ä–∏—Ç—å –¥–µ–π—Å—Ç–≤–∏–µ.

**–û–±—â–∏–π –°–∏–Ω—Ç–∞–∫—Å–∏—Å `for`:**

–¶–∏–∫–ª `for` —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ —Ç—Ä–µ—Ö –æ–±—è–∑–∞—Ç–µ–ª—å–Ω—ã—Ö —á–∞—Å—Ç–µ–π, —Ä–∞–∑–¥–µ–ª–µ–Ω–Ω—ã—Ö —Ç–æ—á–∫–æ–π —Å –∑–∞–ø—è—Ç–æ–π (`;`), –≤–Ω—É—Ç—Ä–∏ –∫—Ä—É–≥–ª—ã—Ö —Å–∫–æ–±–æ–∫:

$$\mathbf{for} \space (\text{–ò–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏—è}; \space \text{–£—Å–ª–æ–≤–∏–µ}; \space \text{–ò–∑–º–µ–Ω–µ–Ω–∏–µ}) \space \{ \space \text{–ë–ª–æ–∫ –∫–æ–¥–∞} \space \}$$

| –ß–∞—Å—Ç—å                | –ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ                                                                                    | –ö–æ–≥–¥–∞ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è                        |
| :------------------- | :-------------------------------------------------------------------------------------------- | :--------------------------------------- |
| **1. –ò–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏—è** | –°–æ–∑–¥–∞–µ—Ç –∏ —É—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç –Ω–∞—á–∞–ª—å–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ **—Å—á–µ—Ç—á–∏–∫–∞** (–æ–±—ã—á–Ω–æ `int i = 0` –∏–ª–∏ `int i = 1`). | **–û–¥–∏–Ω —Ä–∞–∑** –≤ —Å–∞–º–æ–º –Ω–∞—á–∞–ª–µ.             |
| **2. –£—Å–ª–æ–≤–∏–µ**       | –ë—É–ª–µ–≤–æ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –ø—Ä–æ–≤–µ—Ä—è–µ—Ç—Å—è –ø–µ—Ä–µ–¥ **–∫–∞–∂–¥–æ–π** –∏—Ç–µ—Ä–∞—Ü–∏–µ–π.                             | –ü–µ—Ä–µ–¥ –∫–∞–∂–¥—ã–º –∑–∞–ø—É—Å–∫–æ–º –±–ª–æ–∫–∞ –∫–æ–¥–∞.        |
| **3. –ò–∑–º–µ–Ω–µ–Ω–∏–µ**     | –û–±–Ω–æ–≤–ª—è–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ —Å—á–µ—Ç—á–∏–∫–∞ (–æ–±—ã—á–Ω–æ `i++` –∏–ª–∏ `i--`).                                         | –ü–æ—Å–ª–µ **–∫–∞–∂–¥–æ–≥–æ** –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –±–ª–æ–∫–∞ –∫–æ–¥–∞. |

### 2\. –ü–æ—à–∞–≥–æ–≤–∞—è –†–∞–±–æ—Ç–∞ –¶–∏–∫–ª–∞ `for`

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–π –ø—Ä–∏–º–µ—Ä, —á—Ç–æ–±—ã —Ä–∞—Å–ø–µ—á–∞—Ç–∞—Ç—å —á—Ç–æ-–ª–∏–±–æ 10 —Ä–∞–∑ (–Ω–∞—á–∏–Ω–∞—è —Å `i = 1`):

```dart
for (int i = 1; i <= 10; i++) {
  print("Hello World $i");
}
```

1.  **–ù–∞—á–∞–ª–æ:** –í—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –ò–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏—è: `int i = 1`.
2.  **–ü—Ä–æ–≤–µ—Ä–∫–∞:** –ü—Ä–æ–≤–µ—Ä—è–µ—Ç—Å—è –£—Å–ª–æ–≤–∏–µ: `1 <= 10`? (**True**).
3.  **–í—ã–ø–æ–ª–Ω–µ–Ω–∏–µ:** –í—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –±–ª–æ–∫ –∫–æ–¥–∞: `print("Hello World 1")`.
4.  **–ò–∑–º–µ–Ω–µ–Ω–∏–µ:** –í—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –ò–∑–º–µ–Ω–µ–Ω–∏–µ: `i++` (—Ç–µ–ø–µ—Ä—å `i = 2`).
5.  **–ü–æ–≤—Ç–æ—Ä:** –°–Ω–æ–≤–∞ –ü—Ä–æ–≤–µ—Ä–∫–∞ –£—Å–ª–æ–≤–∏—è: `2 <= 10`? (**True**).
6.  ...
7.  **–ü–æ—Å–ª–µ–¥–Ω–∏–π –ø—Ä–æ—Ö–æ–¥:** `i = 10`. –ü—Ä–æ–≤–µ—Ä–∫–∞: `10 <= 10`? (**True**). –í—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –∫–æ–¥. –ò–∑–º–µ–Ω–µ–Ω–∏–µ: `i++` (—Ç–µ–ø–µ—Ä—å `i = 11`).
8.  **–ó–∞–≤–µ—Ä—à–µ–Ω–∏–µ:** –ü—Ä–æ–≤–µ—Ä–∫–∞: `11 <= 10`? (**False**). –¶–∏–∫–ª **–∑–∞–≤–µ—Ä—à–∞–µ—Ç—Å—è**, –∏ –ø—Ä–æ–≥—Ä–∞–º–º–∞ –ø–µ—Ä–µ—Ö–æ–¥–∏—Ç –∫ –∫–æ–¥—É, –∏–¥—É—â–µ–º—É –ø–æ—Å–ª–µ —Ü–∏–∫–ª–∞.

### 3\. –û–ø–µ—Ä–∞—Ç–æ—Ä—ã –ò–Ω–∫—Ä–µ–º–µ–Ω—Ç–∞ –∏ –î–µ–∫—Ä–µ–º–µ–Ω—Ç–∞

–ê–≤—Ç–æ—Ä –≤–≤–æ–¥–∏—Ç **—Å–æ–∫—Ä–∞—â–µ–Ω–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å** –¥–ª—è –∏–∑–º–µ–Ω–µ–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏—è:

- **`i++` (–ò–Ω–∫—Ä–µ–º–µ–Ω—Ç):** –£–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç `i` –Ω–∞ **1**. –≠–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ `i = i + 1`.
- **`i--` (–î–µ–∫—Ä–µ–º–µ–Ω—Ç):** –£–º–µ–Ω—å—à–∞–µ—Ç `i` –Ω–∞ **1**. –≠–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ `i = i - 1`.

#### –°–æ–∫—Ä–∞—â–µ–Ω–Ω—ã–µ –û–ø–µ—Ä–∞—Ç–æ—Ä—ã –ü—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è:

- **`i += 2`:** –£–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç `i` –Ω–∞ **2**. –≠–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ `i = i + 2`.
  - _–ü—Ä–∏–º–µ—Ä –∞–≤—Ç–æ—Ä–∞:_ –ï—Å–ª–∏ –Ω–∞—á–∞—Ç—å —Å `i = 1` –∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `i += 2`, —Ü–∏–∫–ª –±—É–¥–µ—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å—Å—è –¥–ª—è –∑–Ω–∞—á–µ–Ω–∏–π `1, 3, 5, 7, 9` (–≤—Å–µ–≥–æ 5 —Ä–∞–∑), –ø–æ—Å–∫–æ–ª—å–∫—É —Å–ª–µ–¥—É—é—â–∏–π —à–∞–≥ –±—É–¥–µ—Ç `11`, —á—Ç–æ –Ω–∞—Ä—É—à–∞–µ—Ç —É—Å–ª–æ–≤–∏–µ `i <= 10`.

### 4\. –°—Ç–∞–Ω–¥–∞—Ä—Ç –ü—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è: –ù–∞—á–∞—Ç—å —Å –ù—É–ª—è (`i = 0`)

–ê–≤—Ç–æ—Ä –æ—Ç–º–µ—á–∞–µ—Ç, —á—Ç–æ –≤ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–∏, –∏ –≤ Dart, –ø—Ä–∏–Ω—è—Ç–æ –Ω–∞—á–∏–Ω–∞—Ç—å —Å—á–µ—Ç—á–∏–∫ —Ü–∏–∫–ª–∞ —Å **`0`**, –∞ —É—Å–ª–æ–≤–∏–µ –∑–∞–≤–µ—Ä—à–∞—Ç—å **`< N`** (–≥–¥–µ N ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏–π):

```dart
// –°—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–π –ø–æ–¥—Ö–æ–¥: 10 –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏–π (i = 0, 1, 2, ... –¥–æ 9)
for (int i = 0; i < 10; i++) {
  // ... –∫–æ–¥ ...
}
```

**–ü–æ—á–µ–º—É `i = 0`?**
–ë–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä –¥–∞–Ω–Ω—ã—Ö –≤ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–∏ (–º–∞—Å—Å–∏–≤—ã, —Å–ø–∏—Å–∫–∏, —Å—Ç—Ä–æ–∫–∏) –∏—Å–ø–æ–ª—å–∑—É—é—Ç **–∏–Ω–¥–µ–∫—Å–∞—Ü–∏—é, –Ω–∞—á–∏–Ω–∞—é—â—É—é—Å—è —Å –Ω—É–ª—è** (zero-based indexing).

- –ù–∞–ø—Ä–∏–º–µ—Ä, –≤ —Å—Ç—Ä–æ–∫–µ `"Hello"`, –±—É–∫–≤–∞ 'H' –∏–º–µ–µ—Ç **–∏–Ω–¥–µ–∫—Å 0**, –∞ –Ω–µ 1.
- –ú–µ—Ç–æ–¥ **`.substring()`** –¥–ª—è –∏–∑–≤–ª–µ—á–µ–Ω–∏—è —á–∞—Å—Ç–∏ —Å—Ç—Ä–æ–∫–∏ —Ç–∞–∫–∂–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –∏–Ω–¥–µ–∫—Å—ã, –Ω–∞—á–∏–Ω–∞—é—â–∏–µ—Å—è —Å 0.

–ï—Å–ª–∏ –≤—ã –Ω–∞—á–∏–Ω–∞–µ—Ç–µ —Å `i = 0`, –∏ –≤–∞–º –Ω—É–∂–Ω–æ –ø–æ–∫–∞–∑–∞—Ç—å –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—é –Ω—É–º–µ—Ä–∞—Ü–∏—é —Å 1, –≤—ã –º–æ–∂–µ—Ç–µ –ø—Ä–æ—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –≤—ã—Ä–∞–∂–µ–Ω–∏–µ **`i + 1`** –ø—Ä–∏ –≤—ã–≤–æ–¥–µ (–∫–∞–∫ –ø–æ–∫–∞–∑–∞–ª –∞–≤—Ç–æ—Ä):

```dart
// i –±—É–¥–µ—Ç 0, 1, 2, ...
print("Hello World ${i + 1}"); // –í—ã–≤–æ–¥: Hello World 1, 2, 3, ...
```

### 5\. –û–ø–∞—Å–Ω–æ—Å—Ç—å –ë–µ—Å–∫–æ–Ω–µ—á–Ω—ã—Ö –¶–∏–∫–ª–æ–≤ (Infinite Loop)

–ê–≤—Ç–æ—Ä –¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É–µ—Ç —Å–µ—Ä—å–µ–∑–Ω—É—é –æ—à–∏–±–∫—É: **–±–µ—Å–∫–æ–Ω–µ—á–Ω—ã–π —Ü–∏–∫–ª**.

```dart
for (int i = 0; i < 10; i--) { // ‚ùå –û–®–ò–ë–ö–ê
  // ...
}
```

**–û–±—ä—è—Å–Ω–µ–Ω–∏–µ:**

1.  `i` –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å 0.
2.  –£—Å–ª–æ–≤–∏–µ: `i < 10` (–ò—Å—Ç–∏–Ω–Ω–æ).
3.  –ò–∑–º–µ–Ω–µ–Ω–∏–µ: `i--` –¥–µ–ª–∞–µ—Ç `i` —Ä–∞–≤–Ω—ã–º -1, –∑–∞—Ç–µ–º -2, -3 –∏ —Ç.–¥.
4.  –ü–æ—Å–∫–æ–ª—å–∫—É -1, -2, -3... **–≤—Å–µ–≥–¥–∞** –º–µ–Ω—å—à–µ 10, —É—Å–ª–æ–≤–∏–µ **–Ω–∏–∫–æ–≥–¥–∞ –Ω–µ —Å—Ç–∞–Ω–µ—Ç –ª–æ–∂–Ω—ã–º**.
5.  –¶–∏–∫–ª –±—É–¥–µ—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å—Å—è **–≤–µ—á–Ω–æ**, —á—Ç–æ –ø—Ä–∏–≤–µ–¥–µ—Ç –∫ –∑–∞–≤–∏—Å–∞–Ω–∏—é –ø—Ä–æ–≥—Ä–∞–º–º—ã (–∫–∞–∫ –ø—Ä–æ–∏–∑–æ—à–ª–æ —Å –±—Ä–∞—É–∑–µ—Ä–æ–º –∞–≤—Ç–æ—Ä–∞).

**–ü—Ä–∞–≤–∏–ª–æ:** –ö–æ–≥–¥–∞ –≤—ã —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç–µ —Å—á–µ—Ç—á–∏–∫ (`i++`), —É—Å–ª–æ–≤–∏–µ –¥–æ–ª–∂–Ω–æ –ø—Ä–æ–≤–µ—Ä—è—Ç—å **–≤–µ—Ä—Ö–Ω–∏–π –ø—Ä–µ–¥–µ–ª** (`< N`). –ö–æ–≥–¥–∞ –≤—ã —É–º–µ–Ω—å—à–∞–µ—Ç–µ —Å—á–µ—Ç—á–∏–∫ (`i--`), —É—Å–ª–æ–≤–∏–µ –¥–æ–ª–∂–Ω–æ –ø—Ä–æ–≤–µ—Ä—è—Ç—å **–Ω–∏–∂–Ω–∏–π –ø—Ä–µ–¥–µ–ª** (`> N`).

### 6\. –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –¶–∏–∫–ª–∞ `for` –¥–ª—è –†–∞–±–æ—Ç—ã —Å–æ –°—Ç—Ä–æ–∫–∞–º–∏

–ê–≤—Ç–æ—Ä –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç, –∫–∞–∫ —Ü–∏–∫–ª `for` –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è –∏—Ç–µ—Ä–∞—Ü–∏–∏ (–ø–æ–æ—á–µ—Ä–µ–¥–Ω–æ–≥–æ –ø—Ä–æ—Ö–æ–¥–∞) –ø–æ –∫–∞–∂–¥–æ–º—É —Å–∏–º–≤–æ–ª—É –≤ —Å—Ç—Ä–æ–∫–µ.

```dart
String value = "Hello";
// –ó–∞–≤–µ—Ä—à–µ–Ω–∏–µ: I < –¥–ª–∏–Ω—ã —Å—Ç—Ä–æ–∫–∏ (—Ç.–µ. I < 5)
for (int i = 0; i < value.length; i++) {
  // –î–æ—Å—Ç—É–ø –∫ —Å–∏–º–≤–æ–ª—É –ø–æ –µ–≥–æ –∏–Ω–¥–µ–∫—Å—É
  print(value[i]);
}
// –í—ã–≤–æ–¥: H, e, l, l, o (–∫–∞–∂–¥—ã–π —Å–∏–º–≤–æ–ª –Ω–∞ –Ω–æ–≤–æ–π —Å—Ç—Ä–æ–∫–µ)
```

**–ö–ª—é—á–µ–≤–æ–π –º–æ–º–µ–Ω—Ç:** –î–ª—è –¥–æ—Å—Ç—É–ø–∞ –∫ –æ—Ç–¥–µ–ª—å–Ω–æ–º—É —Å–∏–º–≤–æ–ª—É —Å—Ç—Ä–æ–∫–∏ –ø–æ –µ–≥–æ –∏–Ω–¥–µ–∫—Å—É –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è **–∫–≤–∞–¥—Ä–∞—Ç–Ω—ã–µ —Å–∫–æ–±–∫–∏** (`value[i]`).

### 7\. –¶–∏–∫–ª `for...in` (–ü—Ä–æ—Ö–æ–¥ –ø–æ –≠–ª–µ–º–µ–Ω—Ç–∞–º)
