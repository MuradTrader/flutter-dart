–û—Ç–ª–∏—á–Ω–æ\! –í—ã –∑–∞–≤–µ—Ä—à–∏–ª–∏ –∏–∑—É—á–µ–Ω–∏–µ –æ—Å–Ω–æ–≤–Ω—ã—Ö —É—Å–ª–æ–≤–Ω—ã—Ö –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–π –∏ –ø–µ—Ä–µ—Ö–æ–¥–∏—Ç–µ –∫ –±–æ–ª–µ–µ –∫–æ–º–ø–∞–∫—Ç–Ω–æ–º—É –∏ —á–∞—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑—É–µ–º–æ–º—É –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç—É ‚Äî **—Ç–µ—Ä–Ω–∞—Ä–Ω–æ–º—É –æ–ø–µ—Ä–∞—Ç–æ—Ä—É**.

## üí° –¢–µ—Ä–Ω–∞—Ä–Ω—ã–π –û–ø–µ—Ä–∞—Ç–æ—Ä (Ternary Operator)

–¢–µ—Ä–Ω–∞—Ä–Ω—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä (Conditional Operator) ‚Äî —ç—Ç–æ **–∫—Ä–∞—Ç–∫–∏–π —Å–ø–æ—Å–æ–±** –∑–∞–ø–∏—Å–∏ –ø—Ä–æ—Å—Ç–æ–≥–æ —É—Å–ª–æ–≤–Ω–æ–≥–æ –≤—ã—Ä–∞–∂–µ–Ω–∏—è `if-else` –≤ **–æ–¥–Ω–æ–π —Å—Ç—Ä–æ–∫–µ**.

–ê–≤—Ç–æ—Ä –Ω–∞–∑—ã–≤–∞–µ—Ç –µ–≥–æ —Å–æ–∫—Ä–∞—â–µ–Ω–Ω–æ–π –≤–µ—Ä—Å–∏–µ–π `if-else`, –ø–æ–∑–≤–æ–ª—è—é—â–µ–π **–ø—Ä–∏—Å–≤–æ–∏—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –æ–¥–Ω–æ –∏–∑ –¥–≤—É—Ö –∑–Ω–∞—á–µ–Ω–∏–π** –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –∏—Å—Ç–∏–Ω–Ω–æ—Å—Ç–∏ —É—Å–ª–æ–≤–∏—è.

### 1\. –°–∏–Ω—Ç–∞–∫—Å–∏—Å

–¢–µ—Ä–Ω–∞—Ä–Ω—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä ‚Äî —ç—Ç–æ –µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä –≤ Dart (–∏ –≤ –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–µ C-–ø–æ–¥–æ–±–Ω—ã—Ö —è–∑—ã–∫–æ–≤), –∫–æ—Ç–æ—Ä—ã–π —Ç—Ä–µ–±—É–µ—Ç **—Ç—Ä–∏ –æ–ø–µ—Ä–∞–Ω–¥–∞** (–æ—Ç—Å—é–¥–∞ –∏ –Ω–∞–∑–≤–∞–Ω–∏–µ "—Ç–µ—Ä–Ω–∞—Ä–Ω—ã–π").

–ï–≥–æ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –≤—ã–≥–ª—è–¥–∏—Ç —Ç–∞–∫:

$$\text{–£—Å–ª–æ–≤–∏–µ} \space \mathbf{?} \space \text{–ó–Ω–∞—á–µ–Ω–∏–µ\_1} \space \mathbf{:} \space \text{–ó–Ω–∞—á–µ–Ω–∏–µ\_2};$$

- **1. $\text{–£—Å–ª–æ–≤–∏–µ}$**: –õ—é–±–æ–µ –±—É–ª–µ–≤–æ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `true` –∏–ª–∏ `false`.
- **2. `?` (–ó–Ω–∞–∫ –≤–æ–ø—Ä–æ—Å–∞)**: –†–∞–∑–¥–µ–ª–∏—Ç–µ–ª—å. –û–∑–Ω–∞—á–∞–µ—Ç: "–ï—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ –∏—Å—Ç–∏–Ω–Ω–æ...".
- **3. $\text{–ó–Ω–∞—á–µ–Ω–∏–µ\_1}$**: –ó–Ω–∞—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è, –µ—Å–ª–∏ **—É—Å–ª–æ–≤–∏–µ –∏—Å—Ç–∏–Ω–Ω–æ** (`true`).
- **4. `:` (–î–≤–æ–µ—Ç–æ—á–∏–µ)**: –†–∞–∑–¥–µ–ª–∏—Ç–µ–ª—å. –û–∑–Ω–∞—á–∞–µ—Ç: "...–∏–Ω–∞—á–µ (–µ—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ –ª–æ–∂–Ω–æ)...".
- **5. $\text{–ó–Ω–∞—á–µ–Ω–∏–µ\_2}$**: –ó–Ω–∞—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è, –µ—Å–ª–∏ **—É—Å–ª–æ–≤–∏–µ –ª–æ–∂–Ω–æ** (`false`).

### 2\. –ü—Ä–∏–º–µ—Ä—ã –∏ –†–∞–∑–±–æ—Ä

–ê–≤—Ç–æ—Ä –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Å—Ç—Ä–æ–∫–æ–≤–æ–≥–æ –º–µ—Ç–æ–¥–∞ `.startsWith()` –¥–ª—è –±—É–ª–µ–≤–∞ —É—Å–ª–æ–≤–∏—è.

**–†–∞–∑–±–æ—Ä –ø—Ä–∏–º–µ—Ä–∞ –∞–≤—Ç–æ—Ä–∞:**

```dart
String someValue = "Hi"; // –ó–Ω–∞—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –ø—Ä–æ–≤–µ—Ä—è–µ–º

// 1. –£—Å–ª–æ–≤–∏–µ: someValue.startsWith('H')
// (–ú–µ—Ç–æ–¥ .startsWith() –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –±—É–ª–µ–≤–æ –∑–Ω–∞—á–µ–Ω–∏–µ: true –∏–ª–∏ false)

String resultValue = someValue.startsWith('H') ? "Wow" : "Nah";
//                                          ^    ^     ^
//                                         –ï–°–õ–ò  TRUE  –ò–ù–ê–ß–ï

```

**–ü–æ–¥—Ä–æ–±–Ω–æ–µ –æ–±—ä—è—Å–Ω–µ–Ω–∏–µ:**

1.  **–ü—Ä–æ–≤–µ—Ä–∫–∞ —É—Å–ª–æ–≤–∏—è:** –í—ã—Ä–∞–∂–µ–Ω–∏–µ `someValue.startsWith('H')` –ø—Ä–æ–≤–µ—Ä—è–µ—Ç, –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è –ª–∏ —Å—Ç—Ä–æ–∫–∞ `"Hi"` —Å –±—É–∫–≤—ã 'H'. –≠—Ç–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç **`true`**.
2.  **–í—ã–±–æ—Ä –∑–Ω–∞—á–µ–Ω–∏—è:** –ü–æ—Å–∫–æ–ª—å–∫—É —É—Å–ª–æ–≤–∏–µ –∏—Å—Ç–∏–Ω–Ω–æ, –æ–ø–µ—Ä–∞—Ç–æ—Ä –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø–µ—Ä–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, —Å—Ç–æ—è—â–µ–µ –ø–æ—Å–ª–µ `?`, —Ç.–µ. `"Wow"`.
3.  **–ü—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ:** –°—Ç—Ä–æ–∫–∞ `"Wow"` –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π `resultValue`.

–ï—Å–ª–∏ –±—ã —Å—Ç—Ä–æ–∫–∞ –±—ã–ª–∞ `someValue = "Bye"`, —É—Å–ª–æ–≤–∏–µ –≤–µ—Ä–Ω—É–ª–æ –±—ã `false`, –∏ –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è –ø–æ–ª—É—á–∏–ª–∞ –±—ã –∑–Ω–∞—á–µ–Ω–∏–µ, —Å—Ç–æ—è—â–µ–µ –ø–æ—Å–ª–µ `:`, —Ç.–µ. `"Nah"`.

### 3\. –ü—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–∞ –¢–µ—Ä–Ω–∞—Ä–Ω–æ–≥–æ –û–ø–µ—Ä–∞—Ç–æ—Ä–∞

- **–ö–æ–º–ø–∞–∫—Ç–Ω–æ—Å—Ç—å:** –ü–æ–∑–≤–æ–ª—è–µ—Ç –∑–∞–º–µ–Ω–∏—Ç—å 4-5 —Å—Ç—Ä–æ–∫ –∫–æ–¥–∞ (`if`, —Å–∫–æ–±–∫–∏, `else`, —Å–∫–æ–±–∫–∏) –Ω–∞ –æ–¥–Ω—É —Å—Ç—Ä–æ–∫—É.
- **–í—ã—Ä–∞–∂–µ–Ω–∏–µ, –∞ –Ω–µ –û–ø–µ—Ä–∞—Ç–æ—Ä:** –í –æ—Ç–ª–∏—á–∏–µ –æ—Ç `if-else` (–∫–æ—Ç–æ—Ä—ã–π —è–≤–ª—è–µ—Ç—Å—è **–æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–º**), —Ç–µ—Ä–Ω–∞—Ä–Ω—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä —è–≤–ª—è–µ—Ç—Å—è **–≤—ã—Ä–∞–∂–µ–Ω–∏–µ–º** (_expression_). –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –æ–Ω —Å–∞–º –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ, –∏ –µ–≥–æ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç–∞–º, –≥–¥–µ –æ–∂–∏–¥–∞–µ—Ç—Å—è –∑–Ω–∞—á–µ–Ω–∏–µ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –ø—Ä—è–º–æ –≤–Ω—É—Ç—Ä–∏ —Ñ—É–Ω–∫—Ü–∏–∏ `print()` –∏–ª–∏ –∫–∞–∫ —á–∞—Å—Ç—å –¥—Ä—É–≥–æ–≥–æ –≤—ã—Ä–∞–∂–µ–Ω–∏—è).

**–ü—Ä–∏–º–µ—Ä (–∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –±–µ–∑ –ø—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π):**

```dart
// –í–º–µ—Å—Ç–æ —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Å—Ä–∞–∑—É –ø–µ—Ä–µ–¥–∞–µ—Ç—Å—è –≤ print()

// –ï—Å–ª–∏ someValue = "Hi", —Ç–æ:

```

**–ö–ª—é—á–µ–≤–æ–π –º–æ–º–µ–Ω—Ç:** –ó–Ω–∞—á–µ–Ω–∏—è, –≤–æ–∑–≤—Ä–∞—â–∞–µ–º—ã–µ –¢–µ—Ä–Ω–∞—Ä–Ω—ã–º –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–º (`–ó–Ω–∞—á–µ–Ω–∏–µ_1` –∏ `–ó–Ω–∞—á–µ–Ω–∏–µ_2`), **–¥–æ–ª–∂–Ω—ã –±—ã—Ç—å —Å–æ–≤–º–µ—Å—Ç–∏–º—ã** –ø–æ —Ç–∏–ø—É —Å –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, –∫–æ—Ç–æ—Ä–æ–π –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç (–∏–ª–∏ —Å –∫–æ–Ω—Ç–µ–∫—Å—Ç–æ–º, –≤ –∫–æ—Ç–æ—Ä–æ–º –æ–Ω –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è). –í –ø—Ä–∏–º–µ—Ä–µ –≤—ã—à–µ –æ–±–∞ –∑–Ω–∞—á–µ–Ω–∏—è ‚Äî —Å—Ç—Ä–æ–∫–∏ (`"Wow"` –∏ `"Nah"`).
