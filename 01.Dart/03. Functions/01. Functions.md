–ê–≤—Ç–æ—Ä –ø—Ä–µ–¥–æ—Å—Ç–µ—Ä–µ–≥–∞–µ—Ç –æ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –Ω–µ—è–≤–Ω–æ–≥–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ–º–æ–≥–æ —Ç–∏–ø–∞ **`dynamic`** (–∫–æ–≥–¥–∞ –≤—ã –Ω–µ —É–∫–∞–∑—ã–≤–∞–µ—Ç–µ —Ç–∏–ø). –•–æ—Ç—è Dart –º–æ–∂–µ—Ç —ç—Ç–æ –ø–æ–∑–≤–æ–ª–∏—Ç—å, —ç—Ç–æ **–æ—Ç–∫–ª—é—á–∞–µ—Ç –ø—Ä–æ–≤–µ—Ä–∫—É —Ç–∏–ø–æ–≤** (`type checking`) –≤–æ –≤—Ä–µ–º—è –∫–æ–º–ø–∏–ª—è—Ü–∏–∏, —á—Ç–æ –º–æ–∂–µ—Ç –ø—Ä–∏–≤–µ—Å—Ç–∏ –∫ –Ω–µ–ø—Ä–µ–¥—Å–∫–∞–∑—É–µ–º—ã–º –æ—à–∏–±–∫–∞–º –≤–æ –≤—Ä–µ–º—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –ø—Ä–æ–≥—Ä–∞–º–º—ã (runtime errors).

**–ü—Ä–∞–≤–∏–ª–æ:** –í—Å–µ–≥–¥–∞ —è–≤–Ω–æ —É–∫–∞–∑—ã–≤–∞–π—Ç–µ —Ç–∏–ø –≤–æ–∑–≤—Ä–∞—â–∞–µ–º–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è (`int`, `String`, `void` –∏ —Ç.–¥.) –¥–ª—è –æ–±–µ—Å–ø–µ—á–µ–Ω–∏—è –±–µ–∑–æ–ø–∞—Å–Ω–æ—Å—Ç–∏ –∫–æ–¥–∞.

### `const` vs `final`

–ê–≤—Ç–æ—Ä –∫—Ä–∞—Ç–∫–æ –Ω–∞–ø–æ–º–∏–Ω–∞–µ—Ç –æ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö:

- –í—ã –º–æ–∂–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å **`var`** –∏–ª–∏ **`final`** –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞ —Ñ—É–Ω–∫—Ü–∏–∏:
  ```dart
  final name = printName(); // final ‚Äî –∑–Ω–∞—á–µ–Ω–∏–µ –Ω–µ –∏–∑–º–µ–Ω–∏—Ç—Å—è
  var name = printName();   // var ‚Äî –∑–Ω–∞—á–µ–Ω–∏–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –∏–∑–º–µ–Ω–µ–Ω–æ
  ```
- –í—ã **–Ω–µ –º–æ–∂–µ—Ç–µ** –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å **`const`** (–∫–æ–Ω—Å—Ç–∞–Ω—Ç—É) –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞ —Ñ—É–Ω–∫—Ü–∏–∏, –ø–æ—Ç–æ–º—É —á—Ç–æ `const` —Ç—Ä–µ–±—É–µ—Ç, —á—Ç–æ–±—ã –∑–Ω–∞—á–µ–Ω–∏–µ –±—ã–ª–æ –∏–∑–≤–µ—Å—Ç–Ω–æ **–≤–æ –≤—Ä–µ–º—è –∫–æ–º–ø–∏–ª—è—Ü–∏–∏**. –ó–Ω–∞—á–µ–Ω–∏–µ, –≤–æ–∑–≤—Ä–∞—â–∞–µ–º–æ–µ —Ñ—É–Ω–∫—Ü–∏–µ–π, –æ–±—ã—á–Ω–æ –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è —Ç–æ–ª—å–∫–æ **–≤–æ –≤—Ä–µ–º—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è** –ø—Ä–æ–≥—Ä–∞–º–º—ã.

–•–æ—Ç–µ–ª–∏ –±—ã –≤—ã —Ä–∞—Å—Å–º–æ—Ç—Ä–µ—Ç—å, –∫–∞–∫ –ø–µ—Ä–µ–¥–∞–≤–∞—Ç—å –¥–∞–Ω–Ω—ã–µ –≤ —Ñ—É–Ω–∫—Ü–∏–∏ (–ø–∞—Ä–∞–º–µ—Ç—Ä—ã –∏ –∞—Ä–≥—É–º–µ–Ω—Ç—ã)?
